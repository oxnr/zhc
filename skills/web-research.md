# Skill: Web Research

## Purpose
Deep research on any topic using web search and browsing.

## Process
1. Start with broad search queries to map the landscape
2. Narrow down with specific queries
3. Visit top 3-5 sources for detailed information
4. Cross-reference findings across sources
5. Synthesize into actionable insights

## Output Format
```
## Research: [Topic]
### Key Findings
- Finding 1 (source: URL)
- Finding 2 (source: URL)

### Market Size / Opportunity
[Numbers, if applicable]

### Recommendation
[What to do with this information]
```

## Quality Checks
- Minimum 3 sources per finding
- Recency: prefer sources from last 30 days
- Credibility: prefer established sources over blogs
